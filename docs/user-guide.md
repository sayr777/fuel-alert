# Пользовательская документация

## Кто пользуется системой

- **Водители** — сообщают о ситуации на АЗС через Telegram-бота. Это единственный канал приёма
  обращений (веб-формы для отправки нет).
- **Все посетители сайта** — смотрят карту обращений без регистрации.
- **Модераторы** — проверяют обращения, требующие ручного решения (см. [admin-guide.md](admin-guide.md)).

## Как сообщить о ситуации (Telegram-бот)

1. Откройте бота по ссылке (кнопка «Сообщить в боте» на сайте) или введите `/start`.
2. Нажмите кнопку **«📢 Сообщить о ситуации»**.
3. **Выберите тип события** — кнопки строятся из общего справочника: наличие/отсутствие топлива,
   лимит отпуска, большая очередь, завышенная цена, АЗС закрыта, незаконная торговля, недолив,
   контрафакт, мошенничество. Если ни один тип не подходит — выберите **«✏️ Другое»** и опишите
   ситуацию текстом.
4. **Геопозиция** — обязательна, отправляется кнопкой «📍 Отправить геолокацию» в Telegram. Без
   неё обращение не уйдёт дальше.
5. **Фото** — необязательно, до 2 штук. Принимаются как обычные фото, так и файлы (файл сохраняет
   EXIF — время и GPS съёмки — что используется для проверки достоверности). Нажмите
   **«⏭ Пропустить фото»**, если фото нет.
6. **Комментарий** — необязательный короткий текст. Нажмите **«⏭ Без комментария»**, если не
   нужен.
7. **Подтверждение** — бот показывает сводку (тип, место, комментарий, количество фото). Кнопки
   «✅ Отправить» / «❌ Отмена».
8. **Статус** — сразу после отправки бот сообщит один из исходов:
   - **Опубликовано** — сразу видно на карте;
   - **На модерации** — тип события требует ручной проверки, либо автоматика нашла нестыковку
     (например, EXIF-время/координаты фото не совпадают с заявленными);
   - **Дубликат** — рядом (в радиусе ~200 м) и по времени (в пределах ~2 часов) уже есть такое же
     обращение от вас — засчитано как дополнительное подтверждение существующего;
   - **Отклонено** — вне зоны покрытия, слишком старое событие, событие «из будущего» или превышен
     лимit обращений в час.

### Что не публикуется никогда

Наружу (на карту, в API) отдаётся только ваш ник. `telegram_id`, номер телефона и внутренний
`user_id` не публикуются и используются только для антифрода/антиспама.

## Как пользоваться картой

Карта открывается с лендинга (кнопка «Открыть карту») или напрямую.

- **Период** — в шапке справа: быстрые пресеты «24 часа / 3 дня / Неделя» или свой диапазон дат
  через «Период…». По умолчанию показывается последний 24-часовой период.
- **Регион** — селектор над картой переключает область просмотра (все субъекты РФ). Реальные
  данные пока есть только по европейской части России (пилотный регион) — в остальных субъектах
  карта просто переместится туда, но обращений может не быть.
- **Фильтры** (вкладка слева) — тип события, марка топлива, сеть АЗС (включая «Без сети»),
  показывать ли АЗС на карте вообще. У каждого типа события — счётчик, сколько таких обращений
  сейчас в выборке.
- **Список отчётов** и **Легенда** (вкладки справа) — сворачиваемые панели: список карточек
  видимых обращений и расшифровка цветов маркеров.
- **Маркеры** — цвет точки = тип события у обращений или сеть у АЗС; чем больше подтверждений у
  обращения, тем крупнее маркер. Клик по маркеру — попап с деталями (марки топлива, цена, фото,
  описание, ник автора, число подтверждений).
- **Геолокация** — кнопка «Я здесь» в тулбаре карты центрирует карту на вашем текущем местоположении
  (требует разрешения браузера).

## Панель модерации

Доступна по ссылке «Модерация» — только тем, у кого есть токен модератора (см.
[admin-guide.md](admin-guide.md), раздел про доступ модератора). Обычным пользователям показывать
её не нужно.
