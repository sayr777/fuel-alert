# Документация «Топливный Дозор»

Снимок состояния проекта (frontend UI приведён к дизайн-мокапу, бэкенд/бот не менялись в этом заходе).

- [architecture.md](architecture.md) — архитектура, технологии, поток данных
- [screens.md](screens.md) — экраны приложения и их назначение
- [user-guide.md](user-guide.md) — пользовательская документация (водитель + веб-карта)
- [admin-guide.md](admin-guide.md) — установка, конфигурация, эксплуатация, доступ модератора

## Состав репозитория

| Каталог | Назначение |
|---|---|
| `backend/` | FastAPI-бэкенд: приём и выдача обращений, модерация, справочники |
| `bot/` | Telegram-бот на aiogram — единственный канал приёма обращений от водителей |
| `frontend/` | React+TypeScript+Vite веб-карта и панель модерации |
| `backend-rust/` | Экспериментальный Rust-бэкенд (не задействован в основном стеке) |
| `deploy/` | Дополнительный docker-compose для окружения деплоя |
| `docker-compose.yml` | Локальный/продовый запуск: PostGIS, Redis, MinIO, API, бот |
